### Linux
**Build from source** 
Clone the repository
```c
$ git clone "https://github.com/corsum/rhaddon.git"
```
Move to repository directory
```c
$ cd rhaddon
```
Then type the following command
```c
$ ./build
```
**Uninstall**
To unistall Rhaddon from your computer type the command below
```c
$ ./unst
```
